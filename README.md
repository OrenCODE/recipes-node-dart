# Recipes application with TS and Dart
Building a Recipe Application Using Node.js, Express, MongoDB, Dart

<br/>

# Prerequisites
- Node
- nodemon
- express 
- mongoose
- typescript
- cors


<br/>

# Setup
- git clone https://github.com/OrenCODE/recipes-node-dart.git
- npm i
- change mongoDB URI
- npm run dev
- navigate in your browser to localhost:5000
